# LAN-Rock-Paper-Scissors
Rock-Paper-Scissors LAN Game. Implemented using GoLang, Socket.IO, JavaScript, HTML, CSS

```
  go run .\server.go
```
Copy server address and paste it into your browser.
Enjoy the game.
